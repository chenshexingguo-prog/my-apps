一、为什么需要粘塑性正则化？

核心问题：混凝土损伤模型（CDP）或Cohesive单元在软化阶段，应力-应变曲线进入下降段，刚度矩阵可能变为奇异（负定），导致ABAQUS/Standard求解不收敛。

❌ 不使用粘塑性正则化

  • 软化段刚度退化严重
  • 易出现负特征值
  • 迭代不收敛
  • 计算中断

✅ 使用粘塑性正则化

  • 允许应力小幅超出屈服面
  • 粘性阻尼改善数值稳定性
  • 收敛性大幅提升
  • 结果精度影响很小
收敛困难 → 启用粘塑性正则化 → 改善收敛 → 结果验证

二、原理简述（Duvaut-Lions 粘塑性）

ABAQUS采用广义Duvaut-Lions正则化，核心思想：允许应力小幅超出屈服面，通过粘性系数 μ 控制超出程度。

本构方程

ε̇vp = (1/μ) · (ε̇pl − ε̇vp)
参数含义说明
μ（粘度系数）粘塑性松弛时间越小越接近真实解，但改善收敛效果越弱
ε̇vp粘塑性应变率正则化后的有效应变率
ε̇pl无粘性塑性应变率传统塑性模型应变率

μ → 0

退化为标准塑性（无正则化），精度最高但可能不收敛

μ → ∞

完全粘塑性响应，最易收敛但结果可能偏刚

三、CDP模型粘塑性正则化 ⭐（重点）

适用于：混凝土、岩石、砖石等准脆性材料的隐式分析（Standard）。Explicit默认无需设置。

3.1 完整操作流程

1

创建材料 → 进入 CDP 定义界面

路径：Property → Create Material → Mechanical → Plasticity → Concrete Damaged Plasticity

在Property模块中，点击Create Material，命名材料 → 选择Mechanical → 展开Plasticity → 选择Concrete Damaged Plasticity

⚠️ 注意：此选项在ABAQUS/CAE的Material模块中，找到方法：Property module → Create Material → Mechanical → Plasticity → Concrete Damaged Plasticity
2

定义弹性参数

路径：在CDP界面之前，先定义 Elastic → Isotropic

在同一个材料中，选择 Mechanical → Elasticity → Elastic，输入：

  • 弹性模量 E（如：C30混凝土取 30000 MPa）
  • 泊松比 ν（如：0.2）
3

定义塑性参数（膨胀角、偏心率等）

在 Concrete Damaged Plasticity 对话框中填写：

参数建议值说明
Dilation Angle (ψ)30° ~ 35°膨胀角，值越大结构越刚
Eccentricity (ε)0.1偏心率，建议默认0.1
fb0/fc01.16双轴抗压强度比
K2/3屈服面形状参数，0.5~1.0
Viscosity Parameter (μ)0.001 ~ 0.004⭐粘性系数，重点设置！
4

【关键】设置粘性系数（Viscosity Parameter）

界面位置：Concrete Damaged Plasticity 对话框 → 底部 Viscosity Parameters 区域
⚠️ 默认值为0（不做粘塑性正则化）！必须手动输入非零值才能启用。

在 Viscosity Parameter 框中输入数值：

  • 保守起手：0.001
  • 收敛困难：0.003 ~ 0.004
  • 精度优先：0.0001 ~ 0.0005
5

定义受压行为（Compression Hardening）

路径：Concrete Damaged Plasticity → Suboptions → Compression Hardening

需要输入：

  • 屈服应力（Yield Stress）
  • 非弹性应变（Inelastic Strain）
  • 压缩损伤参数 dc（如有）
6

定义受拉行为（Tension Hardening）

路径：Concrete Damaged Plasticity → Suboptions → Tension Hardening

需要输入：

  • 失效应力（Failure Stress）
  • 开裂应变（Cracking Strain）
  • 受拉损伤 dt（如有）
7

定义损伤变量（Compression Damage / Tension Damage）

路径：Concrete Damaged Plasticity → Suboptions → Compression Damage / Tension Damage

可选。如果不定义损伤，CDP将退化为理想塑性模型。

8

创建截面并赋值

路径：Property → Create Section → Category: Solid, Type: Homogeneous

然后将截面分配给相应的Part或区域。

四、Cohesive 单元粘性正则化

适用于：胶接界面、脱粘、分层等问题的收敛改善。

4.1 方法一：通过 Section Controls 设置（推荐⭐）

1

创建 Cohesive Section

路径：Property → Create Section → Category: Other, Type: Cohesive

在 Response 中选择 Traction Separation（牵引-分离本构）

2

定义材料属性

在创建的材料中定义：

  • Elastic（弹性）：Type → Traction，输入法向/切向刚度
  • Damage Initiation（损伤起始）：如 Max Nominal Stress
  • Damage Evolution（损伤演化）：输入断裂能 GIC 等
3

【关键】设置粘性系数

路径：Property → Section Controls → 创建或编辑
⚠️ 关键步骤！在 Section Controls 对话框中，找到 Viscous Regularization 项，输入粘性系数 μ。

Cohesive单元粘性系数参考值：

应用场景建议值说明
水力压裂0.0001 ~ 0.001常用0.0001
胶接界面脱粘0.0001 ~ 0.0005较小值保持精度
复合层合板分层0.001 ~ 0.01根据收敛情况调整
4

分配 Section 给 Cohesive 单元

路径：在Mesh模块中选中Cohesive单元 → Assign Section

4.2 方法二：KEYWORD编辑方式

在 .inp 文件中，直接通过关键字定义：

*COHESIVE SECTION, ELSET=MyCohesiveElements, RESPONSE=TRACTION_SEPARATION
1.0,
*MATERIAL, NAME=Mat-Cohesive
*ELASTIC, TYPE=TRACTION
1000, 385, 385
*DAMAGE INITIATION, CRITERION=MAXS
6.8, 35, 35
*DAMAGE EVOLUTION, TYPE=ENERGY
0.001
*SECTION CONTROLS, VISCOUS_REGULARIZATION=0.0001
MyCohesiveElements
关键行说明：
*SECTION CONTROLS, VISCOUS_REGULARIZATION=0.0001 — 粘性系数为0.0001
也可以写成 *SECTION CONTROLS, VISCOUS_REGULARIZATION=1e-4

5.2 粘性系数对结果的影响

粘性系数 μ行为影响
μ = 0无正则化标准塑性，可能不收敛
μ = 0.0001轻微正则化接近真实解，改善收敛有限

μ = 0.004较强正则化易收敛，结构略偏刚
μ = 0.01+过强正则化结果明显偏刚，不推荐

七、常见问题与解决方案

Q1: 粘性系数是不是越大越好？

❌ 不是！粘性系数过大（如 > 0.01）会导致结构响应明显偏刚，结果失真。建议从 0.001 开始，逐渐调整。

Q2: 收敛仍然困难怎么办？

  • 逐步增大粘性系数（0.001 → 0.003 → 0.005）
  • 减小分析步长（Initial increment / Total step time）
  • 在Step模块中启用 STABILIZE 能量耗散
  • 检查材料参数是否合理（尤其是损伤曲线）

Q3: 如何验证粘性系数的影响？

  • 取 μ = 0（即关闭正则化）运行一次，记录结果
  • 取 μ = 0.001 运行一次
  • 取 μ = 0.004 运行一次
  • 对比三者结果差异，确保在可接受范围内

Q4: Explicit需要设置吗？

不需要。ABAQUS/Explicit 使用显式算法，不存在隐式分析的收敛问题，粘性参数默认设为0即可。

Q5: Cohesive单元粘性正则化和CDP粘塑性有什么不同？

对象混凝土/岩石实体单元粘聚力单元界面
设置位置材料定义中Section Controls
关键字*CONCRETE DAMAGED PLASTICITY*SECTION CONTROLS
用途改善软化段收敛改善界面失效收敛
