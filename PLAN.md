## 四、游戏模块

### 4.1 游戏基类接口

所有游戏必须实现以下接口：

```javascript
class GameBase {
    // 游戏初始化
    init() {}

    // 开始游戏
    start() {}

    // 暂停游戏
    pause() {}

    // 继续游戏
    resume() {}

    // 结束游戏
    end() {}

    // 获取存档数据
    getSaveData() {}

    // 加载存档
    loadSaveData(data) {}

    // 销毁游戏
    destroy() {}
}
```
